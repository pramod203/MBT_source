A basic timetable plugin that allow you to add, delete event and show it on your post or page.

== Description ==

If you are finding a simple basic timetable, this is it.

Features of this plugin:

*	Add,list and delete events in admin panel
*	Output the timetable to any post or page using shortcode '[simpleevent]'
*	Edit timetable.css to your match your blog. You can set different style for past, current and future events.

== Installation ==

1. Upload `simple-event-schedule` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[simpleevent]` shortcode in your templates
